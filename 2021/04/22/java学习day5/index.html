Scanner类的功能: 可以实现键盘输入数据到程序当中

引用类型的一般使用步骤:
1、导包: java.lang包下的类可以不用import（导包快捷键Option+Enter）
2、创建
3、使用

匿名对象

匿名对象就是只有右边的对象，没有左边的名字和赋值运算符

匿名对象每次使用都是一个新的对象，在内存中都是新的地址
使用建议: 如果确定一个对象只需要使用唯一一次，可以使用匿名对象

Random类

Random类用来生成随机数字。

ArrayList 对象数组

数组在使用时有个缺点，一旦创建，程序运行期间长度就不可以改变，为了解决这个问题，可使用ArrayList集合。

泛型: 泛指某一种统一的类型, 只能是引用类型，不能是基本类型

从JDK1.5+开始，支持自动装箱、自动拆箱
自动装箱: 基本类型–> 包装类型(引用类型)
自动拆箱: 包装类型(引用类型) –> 基本类型
