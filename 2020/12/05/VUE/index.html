创建Vue项目

  • 安装vue-cli工具和创建项目
  • 创建项目和安装插件
    执行完vue ui命令后，会打开一个web页面叫vue项目管理器，在打开的vue项目管理器中创建项目，项目创建成功后会自动生成一个标准的vue前端项目目录结构
    之后继续再vue项目管理器中安装需要使用的插件：
    1、组件库: vue-cli-plugin-vuetify
    2、axios: 发送请求

目录结构：
node-modules:存放各种依赖文件和第三方库之类的（本地，不上传git）
public: 存放静态文件
src: 存放源码文件
- assets: 存放图片
- components: 存放组件vue文件（vue做的事情就是将vue文件转化成html文件使得浏览器能够正常解析）
- plugins: 插件文件
- router: 路由，决定页面跳转
- views: 存放页面文件
package.json: 类似于python的requirements文件,用于配置项目所需要的依赖项，之后项目迁移时，运行npm install 命令，就会将文件中的依赖安装到本地，即node-modules中

在项目目录下输入命令npm run serve即可启动项目

1、编写Sign.vue组件文件
组件文件由3部分内容组成 模板、script、样式

HTML部分: 使用标签<template></template>，该部分只能有一个<div></div>标签
JS部分: 使用标签<script></script>，该部分主要用来写js代码逻辑
CSS部分: 使用标签<style></style>，该部分主要用来写css代码
